# Dinosaur game

My attempt at learning Java through recreating the dinosaur game you can play if the Chrome browser cannot find a site.

## Running
### Linux / Mac
    git clone git@github.com:rasmusrim/dinosaur-game.git
    cd dinosaur-game
    ./gradlew run
    
### Windows
    git clone git@github.com:rasmusrim/dinosaur-game.git
    cd dinosaur-game
    .\gradlew run
